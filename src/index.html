Flutter é o kit de ferramentas do Google para construir belas aplicações compiladas nativamente para dispositivos móveis, web, e desktop a partir de uma base de códigos única.

Começar a usar

Vindo de outras plataformas? Documentação: iOS, Android, Web, React Native, Xamarin.

{% asset homepage/icon-development.svg %}

Desenvolvimento Rápido

Pinte seu aplicativo em milissegundos com Stateful Hot Reload. Utilize um grupo de widgets totalmente personalizável para construir interfaces nativas em minutos.

{% asset homepage/icon-ui.svg %}

UI flexível e expressiva

Entregue recursos rapidamente com foco em experiência nativa para o usuário final. A arquitetura em camadas permite a personalização total que resulta em rederização incrivelmente rápida e designs expressivos e flexíveis.

{% asset homepage/icon-performance.svg %}

Desempenho Nativo

Os widgets de Flutter incorporam todas as diferenças críticas entre as plataformas como rolagem, navegação, ícones e fontes, e o seu código Flutter é compilado para código nativo ARM utilizando compiladores nativos de Darts. Assim, Flutter oferece desempenho nativo tanto em iOS quanto me Android.

{% asset homepage/icon-development.svg %}

Fast development

O hot reload do Flutter lhe ajuda a experimentar, construir interfaces, adicionar recursos e corrigir erros fácil e rápido. Tenha a experiência de recarregar seu código em menos de um segundo, sem perder o estado, em emuladores, simuladores e hardwares para iOS e Android.
